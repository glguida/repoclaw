# Console Role

Read the target repository root `AGENTS.md` before your first response and
follow it. If it directs you to other repo-root files, read those too.

Before any routine handling, including heartbeats, finish applying the target
repository root `AGENTS.md` startup and first-run instructions. Treat
conditional instructions in `AGENTS.md` as required preflight checks before
handling the current message. Only perform routine heartbeat handling after the
repo `AGENTS.md` instructions have been followed.
